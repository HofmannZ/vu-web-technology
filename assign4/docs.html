API Reference

Method HTTP request Description Posted data
URIs relative to http://localhost:8080/, unless otherwise noted
GET /products Gets all the products in the inventory.
GET /products/productId Gets a specific product in the inventory.
POST /products Adds a new product to the inventory. category, date, amount, name, location
PUT /products/productId Changes an existing product in the inventory. category, date, amount, name, location
DELETE /products/productId Deletes a product form the inventory.

POST /products

Returns a 201 when the item is succesfully added.

            
{
  "url": "http://localhost:8080/products/4"
}

PUT /products/productId

Returns a 200 when the item is succesfully changed.

            
{
  "category": "Fruit",
  "date": "15.05.1990",
  "amount": "15",
  "name": "Zino",
  "location": "Amsterdam"
}

DELETE /products/productId

Returns a 204 when the item is succesfully deleted.
